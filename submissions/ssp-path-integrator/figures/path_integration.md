A small CI-friendly demo: the true counterclockwise circular trajectory (gray) versus the SSP-PI estimate (black dashed). Parameters are sized for a fast CI run — a 37-dim hexagonal SSP with 200 neurons per VCO, simulated for 0.5 s. The diagonal line at the bottom of the dashed track is the network's brief uninitialised state before the initialisation node activates at *t* = 0 (and steers the SSP to the true starting position at `(0.7, 0)`); after that, the network tracks the trajectory.

With 500–800 neurons per VCO (per the docstring guidance) and slower trajectories, the estimate tracks the truth much more tightly — see Fig. 3 of [Dumont et al. (2023)][paper] for the high-performance regime.
