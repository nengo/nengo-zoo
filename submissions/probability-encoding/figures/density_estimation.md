Demo: estimating a Beta(2, 5) probability density from 500 samples with a 128-dimensional random SSP encoder.

**Left — population mode.** 50 neurons, one per evaluation point on (0, 1), with encoders set to the SSPs of those points. The network is driven by the constant trained mean SSP `mu`; each neuron's mean spike rate over the 2-second run (normalized by `max_rate=50`) is plotted as the density estimate.

**Right — single-neuron mode (swept).** One neuron with encoder = `mu`. The input is a query SSP that sweeps linearly across (0, 1) over the same 2-second run; binning the spike train by the corresponding query value gives the noisier per-bin rate estimate shown. The single-neuron variant fundamentally has less averaging power per query point, hence the residual quantization.

Both modes share the same trained `mu` and FIE normalization `xi` — only the encoder matrix differs. Generated by `examples/example_usage.py`.
